Project Overview

Velora is a web-based photo booth that allows users to capture or upload photos, apply lighting filters, decorate with stickers, and download the final image. Inspired by velvet and aura, Velora delivers a smooth, elegant user experience.

Design Timeline

  • Week 1–2: Concept planning, wireframes, webcam setup
  • Week 3–4: UI refinement, lighting filters, midterm testing
  • Week 5–6: Decoration system, upload + drag/zoom/shrink feature
  • Week 7–8: Branding, bug fixes, final demo prep

Core Features

  • Live Camera: Real-time webcam preview with capture button
  • Lighting Filters: Warm, cool, vintage tones via CSS
  • Decorations: Drag-and-drop stickers, emojis, frames
  • Upload & Edit: Upload photos, drag to reposition, zoom/shrink with gestures
  • Download: Export final image with all overlays

Challenges & Solutions

  • Browser Compatibility: Used WebRTC and tested across Chrome, Safari, Edge
  • Image Merging: Used HTML canvas to combine camera and decorations
  • Drag/Zoom/Shrink: Implemented with JavaScript and CSS transforms

Final Outcome

Velora is a fully functional, elegant photo booth web app ready for demo day. It combines intuitive UI, playful features, and smooth performance to deliver a delightful user experience.
